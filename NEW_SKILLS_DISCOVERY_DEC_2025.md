# New Biomedical Skills & Agents Discovery Report (Dec 2025)

This document tracks newly discovered LLM agents, skills, and tools relevant to the **Universal Biomedical Skills** database. These resources were identified through extensive search of the latest developments in 2024 and 2025.

## üß¨ Genomics & Bioinformatics

### **CRISPR-GPT**
*   **Description**: An LLM agent developed for the automated design of gene-editing experiments.
*   **Application**: CRISPR guide design, off-target analysis.
*   **Status**: Candidate for `Skills/Genomics`.

### **BRAD (Bioinformatics Retrieval Augmented Digital) Assistant**
*   **Source**: [Jpickard1/BRAD](https://github.com/Jpickard1/BRAD)
*   **Description**: Streamlines bioinformatics workflows using RAG. Supports database integration and external code execution.
*   **Key Features**: Biomarker discovery, single-cell analysis with Scanpy.

### **GeneAgent**
*   **Source**: [ncbi-nlp/GeneAgent](https://github.com/ncbi-nlp/GeneAgent)
*   **Description**: Focuses on gene set analysis by accessing domain databases. verify facts to reduce hallucination.

### **CellAgent**
*   **Description**: LLM-driven multi-agent framework specifically designed for automated single-cell data analysis.

## üíä Drug Discovery & Chemistry

### **AgentD**
*   **Source**: [hoon-ock/AgentD](https://github.com/hoon-ock/AgentD)
*   **Description**: Accelerates drug discovery workflows (literature extraction, property prediction, molecule generation).
*   **Integration**: Connects with external APIs and cheminformatics libraries.

### **DrugAgent (CACTUS)**
*   **Source**: [anrohanro/DrugAgent](https://github.com/anrohanro/DrugAgent)
*   **Description**: Integrates generative AI with molecular design tools (REINVENT 4) and databases (DISGENET, ChEMBL).

### **ChemCrow**
*   **Description**: Augmented with 13 expert-designed tools for organic synthesis, drug discovery, and materials design.

### **DEDA (Drug Evaluation and Discovery Agent)**
*   **Source**: [drug-discovery-ai/deda-drug-evaluation-and-discovery-agent](https://github.com/drug-discovery-ai/deda-drug-evaluation-and-discovery-agent)
*   **Description**: "All-in-a-box" AI agent for drug evaluation and discovery. Connects to UniProt, AlphaFold, OpenTargets.

## üè• Clinical & Healthcare

### **LLM Pharma**
*   **Source**: [bab-git/llm_pharma](https://github.com/bab-git/llm_pharma)
*   **Description**: Intelligent clinical trial management system.
*   **Key Capability**: Automates patient eligibility evaluation for clinical trials.

### **ChatGPT-HealthPrompt**
*   **Source**: [atenanaz/ChatGPT-HealthPrompt](https://github.com/atenanaz/ChatGPT-HealthPrompt)

*   **Description**: Tool for anonymization and high-fidelity automated data extraction from medical records.
